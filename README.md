# Fractol
Fractal generator

## 1) set up
git clone the repository and enter `make` in the terminal.

## 2) generate one of the 5 fractals available

- Mandelbrot
- Julia (3 different versions)
- Burningship

## 3) after executing the file, you can click `h` to have the help menu
